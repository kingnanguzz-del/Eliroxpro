# EliroxPro — Phase 1

Transparent trading analytics: signal generation + backtesting.

## API reference

GET /api/signal?symbol=BTCUSDT&interval=15m
GET /api/backtest?symbol=BTCUSDT&interval=15m&entryThreshold=65&stopLossPct=1&takeProfitPct=2
GET /api/price?symbol=BTCUSDT

## Deploy

Build command: npm install
Start command: npm start
